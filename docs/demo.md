# Browser Demo

Try PyOCMF directly in your browser! The interactive demo lets you parse, validate, and verify OCMF data without installing anything.

**[Launch Demo â†’](demo/)**

## Features

- Parse OCMF strings (plain text and hex-encoded)
- Upload and extract OCMF data from XML files
- Validate data structure with Pydantic
- Verify cryptographic signatures (ECDSA)
- 100% client-side - runs entirely in your browser
- Powered by Pyodide (Python compiled to WebAssembly)

## Quick Start

1. Visit the [demo page](demo/)
2. Paste an OCMF string or upload an XML file
3. Optionally provide a public key for signature verification
4. Click "Parse & Validate"

The demo includes example data to help you get started.
